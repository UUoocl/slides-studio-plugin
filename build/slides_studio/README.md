[Slides Studio](https://uuoocl.github.io/slides-studio/) is an HTML page that enables Reveal.js presentations to control Open Broadcast Studio (OBS).  Reveal.js is an open source HTML presentation framework. It's a tool that enables anyone with a web browser to create fully-featured and beautiful presentations for free.

Create slides in HTML, Markdown or with the reveal.js  GUI Slides.com

Slide Studio innovates on the Reveal.js "[speaker view](https://revealjs.com/speaker-view/)" by adding 3 features.
- Connect to the OBS WebSocket Server to remotely control OBS
- a new "Studio" view to configure OBS settings.  
	- Use the Studio view to set the appearance of the slides and camera in OBS 
- a new "Teleprompter" view
	- Slide notes are scrolled like a teleprompter
	- with adjustable the font size and scroll speed
